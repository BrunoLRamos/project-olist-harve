# 1. Por que as vendas estão caindo?
→ o que mudou ao longo do tempo? quais categorias,
regiões ou formas de pagamento mais impactam esse
cenário?

# 2. Quem são os nossos melhores clientes?
→ quais são os perfis de clientes que mais compram,
retornam, avaliam bem e geram mais receita?

# 3. Problemas na entrega estão prejudicando a
satisfação e as vendas?
→ atrasos, distâncias e prazos estimados estão
impactando a nota dos clientes e a recompra?

# 4. Quais categorias de produto mais sustentam (ou
derrubam) a receita?
→ quais categorias são as “estrelas” e quais precisam de
atenção, olhando para volume, preço, fotos e frete?
